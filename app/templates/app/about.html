Why build this site

Hi I'm Jason, a Data engineer and I embarked on building a Django website and deploying it on Azure App Services with a PostgreSQL database to demonstrate my expertise in Python and proficiency with Azure cloud services. Leveraging GitHub for version control and continuous integration/continuous deployment (CI/CD), I successfully established a connection and can quickly and easily push and release changes. Looking ahead, I aim to enrich the site’s functionality by integrating JavaScript to capture user clicks, which will be streamed to Azure Event Hubs for real-time data ingestion. Azure Stream Analytics will then process this data in-flight, facilitating rapid insights. The processed data will find its home in Azure Data Lake Storage for long-term archival. Utilizing Python, I plan to analyze and visualize this data, creating compelling visuals that enhance understanding and drive actionable insights.
